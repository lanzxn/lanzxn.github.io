how to create short cut in ubantu

first, notice this path ~/.local/share/applications where stores short cuts.
in that path, you can see some short cut file that end with .desktop.
.desktop is the file suffix special for ubantu short cut.

Let’s analyse one example:

At the first line, it starts with [Desktop Entry], which means it is a short cut file.
the second line is the version for short cut, its value is always 1.0.
the third line is the file type, its value is always Application, means this file can execute as application.
You can name it using Name property, using Icon property to define the short cut icon, its value is the path where the icon stored.
If you define type as application, you must define exec property, the Exec specifies the execute path where the execute file is. In this example,the execute path for this short cut is /opt/idea/bin/idea.sh.
You can comment short cut using Comment property, also catogory this file using Categories property. In this example, this short cut is in IDE, Development.
If you don’t want to see the terminal when you execute this application, you can set false value for Terminal property.
The field called StartupWMClass is a string that defines in which class the application is grouped in the Unity Launcher at startup. for this example, its start class is called jetbrains-idea

After we analyse this kind of file, now you can create a file ends with desktop in ~/.local/share/applications for yourself.

how to remove short cut in ubantu

Now we know all the short cut files store in ~/.local/share/applications, if you want to remove a short cut, you just delete correspond file which in ~/.local/share/applications
