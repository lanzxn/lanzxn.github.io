how-to-install-jdk-in-ubantu

Install jdk using apt-get

  1. Check which version of the JDK your system is using: java -version
  2. Update the repositories: sudo apt-get update
  3. Install OpenJDK: sudp apt-get install openjdk-8-jdk
  4. Verify the version of the JDK: java -version

Install jdk manually

  1. Go to the oracle website download the JDK: Oracle Java
  2. create a category like /usr/lib/jdk8, unzip to where just created: sudo tar -zxvf xxx.tar.gz -C /usr/lib/jdk8

  7. verify the version of the JDK: java -version

By the way, if you install other verions of JDK, you can input this command to choose the exact version: sudo update-alternatives –config java

After installing the JDK, you may need to set JAVA_HOME to your profile:
export JAVA_HOME=path_to_java_home
