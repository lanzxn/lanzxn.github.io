Installation And Configuration

Installation on Debian based Linux

Installation

  1. update apt-get software source: sudo apt-get update
  2. Execute command in terminal: sudo apt-get install vim
  3. To ensure Vim is correctly installed execute command, it should show as /usr/bin/vim: which vim

Configuration

  1. find current user’s home directory: echo $HOME
  2. Navigate to home directory and create a new file namely .vimrc: touch .vimrc
  3. open the file in text editor, enter text and save it (it will show line numbers when file is opened): set nu

Getting familiar

Start Vim

Use terminal

Open terminal and enter vim command : vim

To close vim, press Esc key followed by colon(:) and q.

In Vim q command stands for quit

Vim modes

Vim supports multiple modes.

Command mode

This is the default mode in which Vim starts. We can enter editor commands in this mode.

NOTE, Any Vim command without colon indicates that we are executing that command in command mode.

Insert mode

You can use this mode to enter/edit text. To switch from default command to insert mode press i key.

It will show current mode in bottom left corner of editor.

You also can use Escape key to switch back to command mode.

Command line mode

This mode is also used to enter commands.

Commands in this mode starts with colon(:).

For instance, in previous section quit command was entered in this mode. We can go to this mode either from command or insert mode.

  • To switch from command mode to this mode just type colon
  • To switch from insert mode to this mode press Escape and type colon

Visual mode

In this mode we can visually select text and run commands on selected sections.

  • To switch from command mode to visual mode type v
  • To switch from any other mode to visual mode first switch back to command mode by pressing Escape, then type v to switch to visual mode

Create new file

  1. Execute command to open Vim: vim
  2. Type command in Vim: :edit message.txt
  3. Switch to insert mode (Enter some text): i
  4. Switch back to command mode: Esc
  5. Save changes to file: :w
  6. Quit Vim: :q

Open file in read-only mode

Use –R option to open file in read-only mode: vim -R message.txt

Edit existing file

  1. Open file using vim: vim message.txt
  2. Switch to insert mode and enter some text there: i
  3. Quit editor without saving changes: :q!
  4. Save changes and quit editor using following command: :wq

Getting Help

Access help manual

Help manual is shipped with Vim editor itself and it is really comprehensive. To access help execute command: :help

Help on specific topic

Help manual will show entire help about Vim. But what if, we are only interested in certain topic. Vim provides command for that as well with syntax: :help <topic-name>

Search help phrase in manual

You can search specific topic only if you know its name. But what if, you don’t know the exact name of help topic. In that case you can use command: :helpgrep <phrase>

Editing

Vim provides many commands which make editing functionality really powerful.

  • Insert text before cursor: i
  • Insert text at the beginning of line: I
  • Append text after cursor: a
  • Append text at the end of line: A
  • Open new line below cursor: o
  • Open new line above cursor: O
  • Subsititute single character: s
  • Subsititute current line: S or cc or C
  • Replace single character: r
  • Replace entire line: R

Navigating

Vim has great support for advanced navigation.

Basic navigation

  • move cursor to left by one position: h
  • move cursor to right by one position: l
  • move cursor to upward direction by one line: k
  • move cursor to downward direction by one line: j

To perform multi-position navigation use number with these commands: <number><command>

  • Move cursor to the beginning of current line: 0
  • Move cursor to the end of current line: $
  • scroll down entire page: ctrl + f
  • scroll up entire page: ctrl + b

Line navigation

  • Jump to the nth line: :n
  • Jump to the start of file: :0
  • Jump to the end of file: :$

Word navigation

  • move cursor to the beginning of the next word: w
  • move cursor to the end of the current word: e
  • move cursor to the beginning of the previous word: b

Revisiting Editing

Buffer

Buffer is temporary memory used by Vim. When we open a file in editor, Vim loads its contents from disk drive. These contents are stored in memory (RAM) whenever we edit a file we are actually editing file from buffer. Once we finish editing and save file that time only buffer contents are transferred to appropriate file.

Swap

Swap area is a file created by Vim to store buffer contents periodically. While editing file our changes may be lost because of any reasons and Vim provides swap files to provide data recovery.

to get actual name of swap file execute command: :swapname

  • Delete character from cursor position: x
  • Delete previous character from cursor position: X
  • Copy single character from cursor position: y
  • Copy current line: Y or yy
  • Paste character after cursor position: p
  • Paste character before cursor position: P
  • Delete word from cursor postion: dw
  • Delete entire line from cursor position: D
  • Delete entire line: dd
  • undo single or multiple actions: u or <n>u
  • undo all execute following command: U
  • redo is apposite action of undo: ctrl+r or OR or :red

Searching

Search is very common action and it is one of the actions we use most of the time.

Search related settings

  • highlight search execute command: :set hlsearch
  • perform incremental search execute command: :set incsearch

Search in current file

Search in forward direction

  • Search expression in forward direction: /<expression>
  • Find next occurrence: n
  • Find previous occurrence: N
  • Repeat previous forward search: //

Search in backward direction

  • Search expression in backward direction: ?<expression>
  • Find next occurrence: n
  • Find previous occurrence: N
  • Repeat previous forward search: ??

Search word under cursor

  • Search next occurrence of current word: *
  • Search previous occurence of current word: #

Search in multiple files

Using vimgrep command we can search expression in multiple files: :vimgrep <expression> <files>

  • Go to next occurrence of expression: :cn
  • Go to previous occurrence of expression: :cN

Working With Multiple Things

Vim is very powerful editor. It supports many advanced features to work with multiple files, buffer and windows.

Multiple files

  • Load new file in buffer for editing: :e or :edit
  • List the files for editing from current directory: :e <tab> or :edit <tab>

Multiple buffers

  • Add file into new buffer: :badd <file>
  • Switch to Nth buffer: :bN
  • Move to the next buffer in buffer list: :bnext
  • Move to the previous buffer in buffer list: :bprevious
  • List all buffers: :buffers
  • Move to the first buffer: :bfirst
  • Move to the last buffer: :blast
  • Load all buffers: :ball

Multiple tabs

  • Open new tab: :tabnew
  • Open new file in tab: :tabnew <file>
  • Close current tab: :tabclose
  • Move to the next tab: :tabnext
  • Move to the previous tab: :tabprevious
  • Move to the first tab: :tabfirst
  • Move to the last tab: :tablast

Multiple windows*

  • Open new window: :new <file>
  • Open file in new window: :new <file>

Markers

Vim supports bookmark feature.

  • create bookmark execute command: m{bookmark-name}
  • Move to the exact location of bookmark: {back-quote}{bookmark-name}
  • Jump to the beginning of bookmark line: '{bookmark-name}
  • List all bookmarks: :marks
  • Delete bookmark: :delmarks {bookmark-name}

Folding

Fold functionality will be useful when file contents large text.

To activate fold use command: :set foldenable, :set foldmethod = indent

  • close fold: zc
  • open fold: zo
  • close all folds: zM
  • Unfold all: zR

Diff

Similar to UNIX diff command we can use Vim to show diff in much better manner. It will show diff in colorful manner.

  • show difference: vimdiff <file> <file> or vim -d <file> <file>
  • perform horizontal split: :diffsplit filename
  • perform vertical split: :diffsplit filename

Navigation in diff mode

  • open scrollbind: :set scrollbind
  • disable scrollbind: :set noscrollbind
  • Switch between diff window: ctrl+w ctrl+w
  • Jump to previous change: [c
  • Jump to next change: ]c
  • Apply changes in current diff window: :diffget
  • Apply change from current pane to another: :diffput

Using Vim As Ide

Synatx highlighting

yntax highlighting is one of the important features of IDE.

To enable syntax highlighting: :syntax on

Smart indentation

To perform auto and smart indentation use command: :set autoindent or :set smartindent

Tips && Tricks

We can instruct Vim to highlight miss-spelled words: :set spell

To change file format to UNIX: :set ff = unix

To change file format to DOS: :set ff = dos

To add line number: :set number
