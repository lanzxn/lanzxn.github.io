how-to-config-environment-variable-in-ubantu

File introduce

ubantu has five path to save environment variables.

  1. /etc/profile: when user first login, ubantu will read this file to set the environment variables for every user.
  2. /etc/environment: the second config file used by system when user login in.
  3. ~/.profile: config environment variable just for current user.
  4. /etc/bashrc: config shell information for every user.
  5. ~/.bashrc: config shell information just for current user.

Config System environment variable

  1. edit system config file: sudo vim /etc/profile
  2. add environment variable to the end of file: export PATH=.:{custom execute path}:$PATH
  3. apply the change: source profile

Config current shell environment variable

  1. edit shell config file: vim ~/.bashrc
  2. add environment variable to the end of file: export PATH=.:{custom execute path}:$PATH
  3. reopen shell

Config current user environment variable

  1. edit user config file: vim ~/.profile
  2. add environment variable to the end of file: export PATH=.:{custom execute path}:$PATH
  3. apply the change: source $HOME/.profile
